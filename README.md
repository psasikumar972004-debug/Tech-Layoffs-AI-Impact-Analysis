# 📊 Tech Layoffs & AI Impact Analysis

## 📌 About the Project

This project performs Exploratory Data Analysis (EDA) on a technology layoffs dataset using Python. The analysis identifies layoff trends, company-wise impacts, industry patterns, funding relationships, and geographical distribution through statistical analysis and data visualization.

---

## 🎯 Project Objective

- Analyze global technology layoffs
- Identify the most affected companies and industries
- Explore layoffs by country and funding stage
- Visualize trends using charts
- Generate business insights from the data

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📋 Project Workflow

1. Import Dataset
2. Data Cleaning

4. Exploratory Data Analysis (EDA)
5. Data Visualization
